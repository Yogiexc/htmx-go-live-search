# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-09

### Added
- Initial release of HTMX Live Search
- Go backend with net/http
- HTMX-powered live search functionality
- Server-rendered HTML templates
- Responsive CSS design with gradient background
- Loading indicator with spinner
- Debounced search (300ms delay)
- In-memory user data storage

### Features
- Real-time search without page reload
- Search by name, email, or city
- Zero custom JavaScript required
- Mobile-responsive design

### Technical
- Go 1.21+ compatibility
- Standard library only (no external deps)
- HTMX 1.9.10
- HTML5 semantic markup
